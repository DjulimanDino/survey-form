# Survey Form

Survey Form is a simple front-end project created with HTML and CSS.  
The purpose of this project is to practice building forms, collecting user input, and structuring a clean web page layout.

## Features
- Text, email, and number input fields
- Select dropdown menu
- Radio buttons
- Checkboxes
- Comment section with textarea
- Clean and minimal styling

## Built With
- HTML5
- CSS3
